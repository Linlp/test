                             光盘使用说明
                            
 
      
一、光盘内容

    
    （1）教学视频：该目录下为本书所涉及的视频讲解文件，用Windows Madia Player或暴风影音等播放软件的最新版本播放即可。
    （2）源代码：该目录下为本书所涉及的源代码。


二、软硬件要求

    操作系统：Windows 2000/XP/2003；
    播放软件：Windows Madia Player或暴风影音等播放软件的最新版本；
    微处理器：奔腾II300MHZ或更高；
    内存：至少64MB；
    硬盘空间:1GB以上；
    其他为常规硬件配置，对显卡、声卡都无特别要求。
